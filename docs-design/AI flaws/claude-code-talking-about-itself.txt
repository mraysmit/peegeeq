Claude Code is primarily a **terminal/command-line tool** — it doesn't have its own IDE built in. The VS Code extension is a separate add-on that wraps it.

A few things to check:

**If you're trying to use the VS Code extension:**
- Open VS Code and go to Extensions (Ctrl+Shift+X)
- Search for "Claude Code" and make sure it's installed and enabled
- You should see a Claude Code panel in your sidebar or bottom panel

**If you're using it from the terminal:**
- Claude Code runs directly in your terminal — you type `claude` to start it
- It works *within* your project directory, reading your files from there
- You'd open your Java/Vert.x project folder in terminal and run `claude` there

**The relationship between the two:**
- The terminal version is the core tool
- The VS Code extension just gives you a panel inside VS Code so you don't have to switch windows
- Both do the same thing underneath
